This tutorial will be a crash course in the basics of how to use MongoDB, as well as an introduction to some of MongoDB's core design principles. We'll start by going over the fundamentals of what MongoDB is, use that as context for starting a simple application, and finish off by showing how to set up MongoDB Replica Sets.

Choosing A Shard Key In MongoDB

Choosing a shard key can be difficult, and the factors involved largely depend on your use case. In fact, there is no such thing as a perfect shard key; there are design tradeoffs inherent in every decision. This talk will discuss those tradeoffs, as well as the different types of shard keys available in MongoDB.
